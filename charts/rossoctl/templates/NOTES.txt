Rossoctl UI

The route for rossoctl UI can be found with the command:

{{- if .Values.openshift }}
```
echo "https://$(kubectl get route rossoctl-ui -n rossoctl-system -o jsonpath='{.status.ingress[0].host}')"
```

{{- else }}
```
{{- $command := `kubectl get httproutes -n rossoctl-system rossoctl-ui -o go-template='http://{{index .spec.hostnames 0}}:8080{{printf "\n"}}'` -}}
{{ $command | indent 2 }}
```
{{- end }}

[tool_catalog]
The Tool Catalog shows all deployed M-C-P tools across namespaces.

[tool_detail]

[agent_navigate]

[agent_chat]
We send a weather query. The agent discovers the tool through the M-C-P Gateway, invokes it, and returns the result.

[agent_response]
The response includes weather data from the tool. In the observability dashboard, the complete trace shows both the language model call and the tool invocation, with timing, payloads, and the resolution path through the M-C-P Gateway.

[end]
